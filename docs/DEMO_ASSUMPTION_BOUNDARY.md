# Demo Assumption Boundary

The default scenario is tuned for demonstration realism. It uses a more conservative acquisition basis, modest rent-growth range, higher exit-cap range, higher operating expense baseline, and current-rate debt assumptions so the visible demo outputs do not look like a sales pitch.

Those inputs are not a market forecast, investment advice, or a validated underwriting recommendation. They are a reasonable local demo starting point for showing how the dashboard handles assumptions, simulation outputs, sensitivity views, exports, and explanation surfaces.

Users should adjust assumptions for any real review and treat the resulting model output as decision-support context, not as a recommendation to buy, sell, finance, or approve an investment.
